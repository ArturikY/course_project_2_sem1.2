# DATABASE

Схемы базы данных и SQL скрипты.

## Файлы

- **schema.sql** - Основная схема базы данных
- **schema_v2.5.sql** - Дополнения к схеме (авторизация и история)
- **add_indexes.sql** - Дополнительные индексы для оптимизации

## Порядок импорта

1. `schema.sql` - создает основную структуру
2. `schema_v2.5.sql` - добавляет таблицы для авторизации
3. `add_indexes.sql` - добавляет индексы для оптимизации

## Таблицы

- **accidents** - данные о ДТП
- **users** - пользователи системы
- **route_history** - история маршрутов пользователей
