# Инструкция по получению API ключа Яндекс.Карт

## Шаг 1: Регистрация
1. Откройте https://developer.tech.yandex.ru/
2. Войдите через Яндекс ID (если нет аккаунта - создайте бесплатно)

## Шаг 2: Создание проекта
1. В кабинете разработчика нажмите **"Создать проект"** или **"Добавить проект"**
2. Заполните форму:
   - **Название**: "Анализ ДТП Москвы" (любое название)
   - **Тип**: "Веб-сервис" или "Веб-сайт"
   - **Домен**: `localhost` (для локальной разработки)

## Шаг 3: Получение ключа
1. После создания проекта откройте его
2. Найдите раздел **"JavaScript API и HTTP Геокодер"**
3. Нажмите **"Создать новый ключ"**
4. Скопируйте API-ключ (строка вида `a1b2c3d4-e5f6-...`)

## Шаг 4: Использование
- Вставьте ключ в файл `index.html` в переменную `YANDEX_API_KEY`
- Бесплатный тариф: 25,000 запросов/день (более чем достаточно для курсового проекта)

## Важно:
- Ключ бесплатный для некоммерческих проектов
- Для локальной разработки можно использовать `localhost`
- Для продакшена нужно указать реальный домен
