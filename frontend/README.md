# FRONTEND

Клиентская часть веб-приложения.

## Файлы

- **index.html** - Главная страница с картой и интерфейсом
- **login.php** - Страница входа в систему
- **register.php** - Страница регистрации

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Яндекс.Карты API 2.1

## API

Все запросы к серверу отправляются на `../backend/`
